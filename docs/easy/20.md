# Valid Parentheses
## Stack
Use a stack check left side of parentheses
**NOTE:**
- `"){"`: when left side is right parenthesis, it must be invalid
- `"((`: Don't have right parenthesis, so it's invalid
### Complexity
- Time: O(n)
- Space: O(n)
