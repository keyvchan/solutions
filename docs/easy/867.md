# Transpose Matrix
## Edge Cases
- The shape of the matrix is not square
## Pre-Allocated Array
### Algorithm
Pre allocate an array of the result size
Iterate through the matrix and transpose the values
### Complexity
Time: O(n*m)
Space: O(n*m)
