# My Calendar I



## Merge Interval



### Algorithm

- Checking overlap, if overlap, return false
- Insert the new interval into the list
    - If end == interval.start, merge, 
    - If start == interval.end, merge,
    - Insert it
    - Sort the list


### Complexity

- Time: O(n)
- Space: O(1)
