# Merge Intervals


## Merge


### Algorithm

Sort the intervals by start value.
Loop through the sorted array:
- If the current interval overlaps with the previous one, merge them into one.
- Otherwise, add the current interval to the result.

### Complexity

- Time: O(n)
- Space: O(1)
