# Search in Rotated Sorted Array
## Two binary search
### Algorithm:
Find the pivot point by binary search, find the target based on the pivot point
**NOTE**:
- The shifted index: index = (index + shift) % len(nums)
### Complexity:
- Time: Two Binary Search O(2log(n))
- Space: O(1)
