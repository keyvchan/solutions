# [MEDIUM] Merge Triplets to Form Target Triplet


## Merge


### Algorithm

Merge all triplets into one triplet.
When merging, we need to check if the sum of the element in triplet is equal to the target.

### Complexity

- Time: O(n)
- Space: O(1)
