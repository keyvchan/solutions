# The k-th Lexicographical String of All Happy Strings of Length n



## Backtracking



### Algorithm

- Generate all happy string that length is n.
- Sort the happy string.
- Reutrn the k-th lexicographical string.


### Complexity

- Time: O(n^2 * k)
- Space: O(n^2 * k)
