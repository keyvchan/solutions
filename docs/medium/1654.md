# Minimum Jumps to Reach Home



## DFS + Memoization



### Algorithm

In each position, we have multiple choices:
- If last is forward, we can jump forward and backward
- If last is backward, we can jump backward


### Complexity

- Time: O(n)
- Space: O(n)
