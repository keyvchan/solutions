# Pow(x, n)



## Edge Case

- If n is 0, return 1
- If n is 1, return x
- If n is `INT_MAX`, return infinity


## Divide and Conquer



### Algorithm

- Base Case:
    - If n is 0, return 1
    - If n is 1, return x
    - If n is `INT_MAX`, return infinity
- Recursive Case:
    - If n is even, return Pow(x, n/2) * Pow(x, n/2)
    - If n is odd, return Pow(x, (n-1)/2) * Pow(x, (n-1)/2) * x


### Complexity

- Time: O(log n)
- Space: O(log n)
