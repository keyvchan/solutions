# Range Sum Query 2D - Immutable
## Edge Case
- col1 == 0
## Combine Sum
Calc combine sum of every column
The range sum should be `sum[i][col2] - sum[i][col1 - 1]`
### Complexity
- Time: Calculate sum of every column O(n), do range sum O(r), r is row2 - row1
- Space: O(n)
