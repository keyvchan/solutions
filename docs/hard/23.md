# Merge k Sorted Lists



## Brute Force



### Algorithm

Find the smallest element in the lists, and add it to the result list.
Remove the smallest element from the list.
Repeat until all the lists are empty.


### Complexity

- Time: O(nk^2)
- Space: O(1)
